For developers & agents

Register & verify via terminal

# Install the id.cv CLI
npm install -g @id/cv

# Register your domain
idcv register --name "John Doe"

# Verify via LinkedIn
idcv verify --method linkedin

# Verify via Self.xyz
idcv verify --method self

# Check verification status
idcv status johndoe.cv

# Lookup any .cv domain
dig +short TXT johndoe.cv

Works in any terminal, IDE, or agent environment.

Use in your agent / MCP server

# Verify an agent's identity
dig +short TXT agent.johndoe.cv

# Single domain lookup
curl https://id.cv/api/lookup/johndoe.cv
